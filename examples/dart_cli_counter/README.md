# dart_cli_counter

CLI example using `opentui`.

## Run

```bash
cd /Users/jaichang/Documents/GitHub/flutter_opentui
dart run melos bootstrap
dart run examples/dart_cli_counter/bin/main.dart
```

Controls:

- `+`: increment counter.
- `-`: decrement counter.
- `Tab`: switch focus between input/select.
- `Ctrl+C`: exit.
